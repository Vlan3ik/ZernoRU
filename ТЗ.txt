Разработка веб-маркетплейса Zerno.com для торговли зерновой продукцией и сельхозтехникой в аграрном секторе Смоленской области.

1) Функционал маркетплейса

Покупка зерновой продукции: Пользователи (покупатели) просматривают каталог проверенных продавцов (с верификацией по ИНН, сертификатам качества и отзывам). Фильтры: тип зерна (пшеница, ячмень и т.д.), объем, цена, регион. Корзина с оплатой онлайн (банковские карты, СБП) или по счету для юрлиц. 
Регистрация продавца: Фермеры/производители регистрируются с верификацией (ИНН, ОГРН, фото документов). После одобрения получают личный кабинет для загрузки лотов.

Логистика и доставка: Опции самовывоза или доставки продавцом/через партнеров (Яндекс.Доставка для агро или частные организации). Сервис рассчитывает цену доставки.

Процесс размещения лота продавцом: В ЛК продавец создает заявку: описание товара (тип зерна, сорт, объем в тоннах, проба/качество), цена (фиксированная или аукцион), наличие транспорта (собственный/аренда, с опцией расчета). Обязательные документы для налоговой: сертификат соответствия (ФГИС "Меркурий"), декларация о соответствии, ИНН/КПП, договор на хранение. После публикации — уведомления о просмотрах/заявках.

Торговля сельхозтехникой: Аналогичный функционал для покупки/продажи (тракторы, комбайны). Фильтры: бренд, состояние (новое/б/у), год выпуска.


2) Платная подписка ZernoPlus

2.1)Доступ к обучающим материалам: Ежемесячная/годовая подписка открывает библиотеку: видео/курсы по агрономии (севооборот, удобрения), вебинары от экспертов Смоленской аграрной академии. Персонализированные рекомендации на основе профиля фермы (AI-анализ по региону).

2.2)Аналитика рынка: Встроенные инструменты для анализа:

Нормализованный вегетационный индекс (NDVI): Карты на основе спутниковых данных (от Росреестра/ЕСА) для оценки урожайности полей.

Сельскохозяйственный стресс-индекс (SSI): Мониторинг засухи/болезней по регионам.

Прогнозы цен: Модели на базе ML (данные Цены РФ + историка) для внутреннего рынка (пшеница +5% к сезону).

Балансы спроса/предложения: Отчеты по объемам (экспорт/импорт, склады) с графиками и алертами.
